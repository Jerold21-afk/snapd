# PhiloQuest (Mobile-First Demo)

A playful, gamified philosophy learning app prototype for mobile devices.

## Features

- **Daily Quest system** with streak tracking.
- **XP + level progression** for completing activities.
- **Quick multiple-choice challenges** from different philosophy topics.
- **Philosopher explorer** with character-like cards.
- **Reflection journal** that saves locally on the device/browser.

## Run

Open `index.html` in any modern mobile browser (or desktop browser in mobile view).

No backend or build step is required.
